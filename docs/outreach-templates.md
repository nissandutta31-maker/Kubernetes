# Outreach System (5–10 messages per week)

Use these templates for targeted LinkedIn or email outreach. Customize every message — never mass-paste.

## Weekly checklist

- [ ] Identify 5–10 people (mix of university recruiters and DGX Cloud / platform engineers)
- [ ] Personalize opening line with their team, product, or a recent post
- [ ] Link GitHub repo + demo video (once recorded)
- [ ] Ask for **feedback**, not a job
- [ ] Log contacts in a spreadsheet (name, date, follow-up date)
- [ ] One polite follow-up after 7–10 days if no reply

## LinkedIn message template

```
Hi [Name] — I'm [Your Name], [year] at [School], interested in [specific team/product, e.g. DGX Cloud platform work].

I built a reproducible K8s runtime demo (Go + Docker + Kind) with automated health checks
and an optional GPU scheduling manifest: https://github.com/nissandutta31-maker/Kubernetes

Demo video: [YOUR_DEMO_VIDEO_URL]

I know [season] intern postings may be closed. If you're open to it, I'd appreciate 15 minutes
of feedback on whether this direction fits platform intern work at NVIDIA, or who to follow
for the next cycle.

Thanks for your time.
```

## Follow-up (once only)

```
Hi [Name] — following up briefly on my note from [date]. No worries if you're busy.
I updated the repo based on [any feedback / CI + HTTP health checks] since then.
Happy to share a 90-second demo if useful. Thanks again.
```


## Referral ask (after positive conversation)

```
Thanks again for the feedback on [specific item]. Would you be comfortable referring me
when the next intern req opens if I address [X and Y] first? I plan to apply on the
official portal the day it posts.
```

## Rules

- Max one follow-up per person
- Do not argue if someone declines
- Turn every reply into a concrete repo or resume improvement
