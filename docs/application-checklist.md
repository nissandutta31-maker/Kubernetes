# Next-Cycle Application Checklist

Use this when NVIDIA intern requisitions open on the [NVIDIA careers portal](https://www.nvidia.com/en-us/about-nvidia/careers/).

## Before the req opens

- [ ] Repo runs end-to-end: `make all` → `make port-forward` → `curl /health`
- [ ] Demo video recorded and linked in README
- [ ] Resume bullet updated (see [`profile-materials.md`](profile-materials.md))
- [ ] LinkedIn Featured section updated with repo + video
- [ ] 20+ targeted outreach conversations logged (goal over 2–3 months)
- [ ] At least one engineer's feedback implemented in the repo

## Day the req opens

- [ ] Apply on official portal within 24 hours
- [ ] Use same one-line pitch as resume summary
- [ ] Cover letter: mention project + honest late-cycle framing (see below)
- [ ] Notify anyone who offered to refer you
- [ ] Add GitHub and demo video links in application if free-text fields exist

## Cover letter snippet (if you missed prior deadline)

```
I missed the [prior season] application window but used the time to build and harden
a project aligned with DGX Cloud-style Kubernetes runtime work: reproducible deploy
(Go, Docker, Kind), automated health validation, and a documented GPU scheduling manifest.
I'm targeting this cycle and welcome feedback on remaining gaps.
```

## After applying

- [ ] Send a short note to warm contacts: "Applied to req [ID/title] — thanks for any guidance"
- [ ] Continue weekly outreach until interview or cycle closes
- [ ] Do not spam recruiters about application status more than once

## Parallel credibility (optional)

- NVIDIA Deep Learning Institute or CUDA fundamentals course
- Small open-source PR to a K8s or inference-related project
- Public notes from GTC sessions relevant to DGX Cloud
